# TripleTen Projects
Projects I worked on while attending the Data Analysis TripleTen Bootcamp.

| Project number | Title | Description |
| :-----------: | ----------- |----------- |
| 1 | EDA Project: Instacart | Cleaned and prepared Instacart consumer data to generate insights into shopping habits. |
| 2 | Video Game Sales Analysis | Identified patterns to determine whether a video game is likely to succeed, aimed at preparing marketing campaigns for the following year. |
| 3 | Supervised Learning Project | Used machine learning to predict whether a customer will leave a bank soon, evaluating performance with AUC-ROC and F1 score. |
| 4 | Integrated Project | Determined the best geographical area for an oil drilling company using ML models, evaluated with sMAPE. |
| 5 | Time Series Project | Built a time series ML model to accurately predict the number of taxi orders in the next hour, evaluated using RMSE. |
